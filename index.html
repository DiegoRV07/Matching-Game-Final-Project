Description

This game is the Matching Game! The objective of this game is to match the cards that have the same picture. Once you click a card, the card will flip and reveal an image. If you are sucssessful in matching two cards together, you get a point! If you get 6 points you win the game and get to restart with a new shuffled game.

Features

  1. Timer Count, Score Keeper, Move Detector
  2. Shuffle on Start
  3. Flippable cards
  4. Forum to report problems, (WIP)
